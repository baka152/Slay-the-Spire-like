# Slay-the-Spire-like

一个用 Unity 开发的类杀戮尖塔风格的卡牌游戏。

## 项目说明

本仓库仅包含 `_Scipts` 文件夹中的 C# 脚本代码。

### 脚本分类

- **Models** - 游戏数据模型（卡牌、效果、目标模式等）
- **Systems** - 核心游戏系统逻辑
- **GameActions** - 具体游戏行为
- **UI** - 游戏界面
- **Data** - 数据处理
- **Creators** - 对象生成工具
- **Views** - 视图逻辑
- **General** - 通用工具代码



## 环境要求

- Unity 2022.3 或更高版本
- C# 10+
